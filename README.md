# 🛍️ Exploratory Data Analysis of Fashion Retail Sales and Returns

## 📌 Project Overview

This project performs an end-to-end Exploratory Data Analysis (EDA) on a fashion boutique dataset to uncover business insights related to pricing, inventory, returns, and customer ratings.

The analysis was conducted using Python and focuses on identifying patterns that support data-driven decision-making.

---

## 🎯 Project Objective

- Perform data cleaning and preprocessing
- Analyze brand and category performance
- Evaluate pricing and markdown strategies
- Study return behavior
- Generate actionable business insights
- Create meaningful visualizations

---

## 🛠️ Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📊 Dataset Information

- 2176 rows
- 14 columns
- Includes:
  - Product ID
  - Category
  - Brand
  - Season
  - Size
  - Price details
  - Stock quantity
  - Customer rating
  - Return information

---

## 🔍 Key Analysis Performed

### 1️⃣ Pricing Analysis
- Average Original Price: 97.20
- Average Current Price: 85.50
- Strong positive correlation (0.92) between original and current price

### 2️⃣ Brand Performance
- Zara has the highest product count
- Gap has the lowest inventory footprint

### 3️⃣ Seasonal Distribution
- Summer has the highest product volume
- Winter has the lowest product count

### 4️⃣ Return Rate Analysis
- Outerwear and Bottoms have the highest return rates (~16%)
- Tops have the lowest return rate (~12.8%)

### 5️⃣ Revenue Analysis
- Outerwear generates the highest revenue
- Accessories generate the lowest revenue

### 6️⃣ Customer Rating Insights
- Average rating: 3.0
- Mixed satisfaction levels across products

---

## 📈 Business Insights

- The store follows a mid-range pricing strategy.
- High returns in Bottoms and Outerwear suggest fit/quality issues.
- Accessories are low-risk, steady-selling products.
- Markdown percentage does not significantly impact return rate.

---

## 💡 Recommendations

- Investigate sizing and quality issues in high-return categories.
- Improve customer engagement to increase ratings.
- Optimize discount strategy for slow-moving inventory.
- Focus on high-revenue categories with improved quality control.
